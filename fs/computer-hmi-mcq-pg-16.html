Question 451 :
Which of the following does not suits for Title bar:

  1. Clearly identify the source of the message
  2. Do not include an indication of message type
  3. Use mixed case in the headline style
  4. Compulsorily use numeric value
  Compulsorily use numeric value

Question 452 :
Regularity in screen design is achieved by _________________for screen elements.

  1. Establishing standard, consistently spaced column and row starting points
  2. Establishing relation between different elements
  3. Establishing the connection between the components
  4. Establishing the link for elements of the design.
  Establishing standard, consistently spaced column and row starting points

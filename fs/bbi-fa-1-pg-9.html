Question 241 :

  4. None of the above

  4. None of the above

Gross Profit = (Sales + Closing Stock) - (Opening Stock + Purchases + _____ ).

Under which system only cash transactions are recorded.

  1. Barter System
  2. Cash System
  3. Accrual System

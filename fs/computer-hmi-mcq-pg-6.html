Research has found that _____and more accurately than text and that the graphical attributes of icons.

  2. faster learning
  3. faster use and problem solving

The operating system creates _____ from the physical computer

  1. Virtual space
  2. Virtual computers
  3. Virtual device
  4. Virtual disk

  1. Throw away
  2. Incremetal

  3. an organizational rule

Question 158 :
Windows that appear change contents, and close as a direct result of user actions in the task domain are called

  1. Contextual windows
  2. WIMP
  3. Direct action windows
  4. User controlled windows
  WIMP

Question 159 :
Which of the following instances illustrates ‘cognitive walkthrough’?

  1. Talk to the users about their needs
  2. Consult the experts
  3. Imagine yourself as the user and think from their perspective
  4. Pass the model to other
  Imagine yourself as the user and think from their perspective

Question 160 :
____________ when selected by user will invoke specific operations and are placed in an isolated region within the display

  1. Buttons
  2. Menus
  3. Pointers
  4. Windows
  Buttons

Question 161 :
if information displayed on a screen is packed or crowded,_____

  1. use color to provide visual groupings
  2. color code these kinds or qualities for contrast
  3. color code the different parts to focus selective attention on each in turn
  4. color code the types of status that the information may possess
  use color to provide visual groupings

Which of the following is not a user interface design process?

  2. Interface design
  3. Knowledgeable, frequent users
  4. Interface validation

Question 164 :
_____ is especially suited to showing things that move or change over time

  1. audio
  2. video
  3. diagram
  4. drawing

Question 165 :
decorative graphics is used to _____

  1. identify the links that may be followed
  2. provide visual appeal and emphassis
  3. show how things or process works
  4. illustrate items mentioned in the text

Question 166 :
What is the primary interactive method of communication used by humans?

  1. reading
  2. writing
  3. speaking
  4. listening
  speaking

Question 167 :
Engineering a user interface involves a quantitative engineering style of working in which measurements are used to judge the quality of interface. Hence ____________ is appropriate.

  1. Usability testing
  2. Field study
  3. Predictive evaluation
  4. DECIDE framework
   Usability testing

Question 168 :
Which of the following is true for a mobile design?

  1. People respond to the visual aesthetic.
  2. The colors used, square and rounded corners, gradient and flat backgrounds help improve user experience.
  3. Great information design is a great visual design.
  4. User’s experience is completely dependent on all the facets of the mobile information architecture.
  User’s experience is completely dependent on all the facets of the mobile information architecture.

Question 169 :
Which of the following is not a beneficial attribute of well-designed systems that use direct manipulation?

  1. Novices can learn basic functionality quickly
  2. Experts can work rapidly to carry out a wide range of tasks
  3. Knowledgeable intermittent users can retain operational concepts
  4. User actions are permanent and cannot easily be undone
  User actions are permanent and cannot easily be undone

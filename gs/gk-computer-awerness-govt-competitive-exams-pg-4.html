Question 91 :
What does storage unit provide?

Question 92 :
What is a firmware?

  1. A program started in RAM.
  2. A program started in a file.
  3. A program started in boot sector.

Question 97 :
Which of the following is true?

  1. A byte is a single digit in a binary number.

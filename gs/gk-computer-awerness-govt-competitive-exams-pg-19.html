Question 541 :
The communication line between CPU memory and Peripherals is called a _________

  1. Registers
  2. Mother Board
  3. Bus
  4. None of the Above
   Bus

Question 542 :
The home page of a website is

  1. the largest page
  2. the last page
  3. the first page
  4. the most colourful page
   the first page

Question 543 :
For selecting and highlighting, which of the following is generally used?

  1. Icon
  2. Mouse
  3. Plotter
  4. Floppy disk
   Mouse

Question 544 :
When you turn on the computer, the boot routine will perform this test ........

  1. RAM test
  2. Disk drive test
  3. Memory test
  4. Power - on self-test
   Disk drive test

Question 545 :
The following are all mobile computing devices, except

  1. notebook computers
  2. Cellular phone
  3. digital scanners
  4. Personal Digital Assistant
   digital scanners

Question 546 :
What is the main folder on a storage device called?

  1. Platform
  2. Interface
  3. Root directory

  1. Handouts

Which key is used in combination with another key to perform a specific task?

  1. Ctrl + X
  2. Ctrl + Y
  3. Ctrl + Z

Which Flip Flop is used to store data in registers?

  1. D Flip Flop
  2. JK Flip Flop
  3. RS Flip Flop
  4. None of the Above

Question 551 :
This type of software works with end users, application software and computer hardware to handle the majority of technical details

  1. Communications software
  2. Application software
  3. Utility software
  4. System software
   System software

Question 552 :
VDU is also called

  1. Screen
  2. Monitor
  3. Both 1 - 2
  4. printer

Question 553 :
In a relational database, this is a data structure that organizes the information about a single topic into rows and columns.

  1. Block
  2. Record
  3. Tuple
  4. Table

Question 554 :
A collection of related files is called a?

  1. character
  2. field
  3. database
  4. adjusting
   database

Question 555 :
One nibble is equal to

  1. 2 bits
  2. 4 bits

Question 556 :
The command used to print a file

  1. Shift+P
  2. Ctrl+P
  3. Alt+P
  4. Esc+P
   Ctrl+P

Question 557 :
Network components are connected to the same cable in the_____ topology.

  1. Star
  2. Ring
  3. Bus
  4. Mesh
   Ring

Question 558 :
What is the Super Computer developed in India

  1. CRAY
  2. PARAM
  3. Both (a) and (b)
  4. None of the above
   PARAM

Question 559 :
First generation computer systems used

  1. Transistors
  2. Vacuum Tubes
  3. Both (a) and (b)
  4. None of the above
   Vacuum Tubes

Question 560 :
The primany job of an operating system is to manage

  1. commands
  2. users
  3. programs
  4. resources
   resources

Question 561 :
Electronic Delay Storage Automatic Computer(EDSAC) was invented by _________

  1. Herman Hollerith
  2. JW Mauchy
  3. John Von Neumann
  4. None of the Above
   John Von Neumann

Question 562 :
Sending an email is similar to

  1. picturing an event
  2. narrating a story
  3. writing a letter
  4. creating a drawing
   writing a letter

Question 563 :
________ is a printer language that uses English phrases and programmatic constructions to describe the appearance of a printed page to the printer

  1. Postscript
  2. PCL
  3. Both (A) and (B)
  4. None of the Above
   Postscript

Question 564 :
The output quality of a printer is measured by

  1. dots per sq inch
  2. dots per inch
  3. Dots printed per unit
  4. dots printed per line
   dots per sq inch

Question 565 :
The load instruction is mostly used to designate a transfer frommemory to a processor register known as_________

  1. Accumulator
  2. Instruction Register
  3. Program counter
  4. Memory address Register

Question 566 :
In a ring topology, the computer in possession of the _____ can transmit data?

  1. packet
  2. data
  3. access method
  4. token
   token

Question 567 :
A device that operates under the control of another device is called?

  1. Stem
  2. Slave
  3. Simulator
  4. Emulator

Question 568 :
What is backup?

  1. Adding more components to your network
  2. Protecting data by copying it from the original source to a different destination
  3. Filtering old data from the new data
  4. Accessing data on tape

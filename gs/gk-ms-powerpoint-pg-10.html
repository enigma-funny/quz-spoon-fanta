Question 271 :
Which of the following is the default page setup orientation for notes pages, outlines and handouts?

  1. Vertical
  2. Landscape
  3. Portrait
  4. None of above
   Portrait

Question 272 :
Material consisting of text and numbers is best presented as

  1. A table slide
  2. A bullet slide
  3. A title slide
  4. All of above
   A table slide

Question 273 :
An organization has a president vice president, managers and supervisors. On what level of an organization chart are the vice presidents?

  1. Fourth level
  2. Third level
  3. Second level
  4. First level
   Second level

Question 274 :
The effect applied to display when slides changes in slide show view is

  1. Slide Animation
  2. Custom Animation
  3. Custom Transition
  4. Slide Transition
   Slide Transition

Question 275 :
Which of the following is not a transition effect?

  1. Blinks diagonal
  2. Dissolve
  3. Fade through black
  4. Blinds vertical
   Blinks diagonal

Question 276 :
Animation schemes can be applied to..... in the presentation

  1. All slides
  2. Select slides
  3. Current slide
  4. All of the above
   All of the above

Question 277 :
To insert slide numbers

  1. Insert a text box and select Insert ? Page Number
  2. Insert a textbox and select Insert ? Number ? PageNumber
  3. Choose Insert > > Slide Number
  4. Insert n new text box and select Insert > > slide Number
   Insert n new text box and select Insert > > slide Number

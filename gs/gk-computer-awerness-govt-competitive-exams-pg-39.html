Question 1141 :
A basic instruction that can be interpreted by computer generally has ________

  1. An operand and an address
  2. decoder and an accumulator
  3. Sequence register and decoder
  4. None of the Above
   An operand and an address

Question 1142 :
In MICR, C stands for

  1. Code
  2. Colour
  3. Computer
  4. Character
   Character

Question 1143 :
Computer ___________is whatever is typed, submitted or transmitted to a computer system.

  1. input
  2. output
  3. data
  4. circuitry
   input

Question 1144 :
What does storage unit provide?

  1. Place for typing data
  2. Storage for information and instruction
  3. Place for printing information

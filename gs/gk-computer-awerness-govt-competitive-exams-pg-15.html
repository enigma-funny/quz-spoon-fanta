Question 421 :
Which of the following is/are example(s) of Word Processors?

  1. Microsoft Word
  2. WordPerfect
  3. Both (A) and (B)
  4. None of the above
   Both (A) and (B)

Question 422 :
To make a notebook act as a desktop model, the notebook can be connected to a __________ which is connected to a monitor and other devices.

  1. bay
  2. docking station
  3. port
  4. network
   docking station

Question 423 :
What type of error does a memory parity error create?

  1. A fatal exception error
  2. An NMI error
  3. A corrupt Windows operating system file
  4. A GPF error
   An NMI error

Reusable optical storage will typically have the acronym

  1. CD
  2. DVD
  3. ROM
  4. RW

Question 425 :
The primary aim of computer process is to convert the data into ______

The simultaneous execution of two or more instructions is called

  1. sequential access
  2. reduced instruction set computing
  3. multiprocessing
  4. disk mirroring

Question 429 :
The ________ tells the computer how to use its components.

  1. utility
  2. network
  3. operating system
  4. application program
   operating system

Question 430 :
What process should be used to recall a document saved previously?

  1. Copy
  2. Save
  3. Retrieve
  4. Enter

Question 431 :
This component is required to process data into information and consists of integrated circuits.

  1. Hard disk
  2. RAM
  3. CPU
  4. ROM
   CPU

Question 432 :
__________ transmits one bit of data through a single wire

  1. Serial Port
  2. Fire wire
  3. Parallel Port
  4. None of the Above
   Serial Port

Question 433 :
What is a back up ?

  1. Restoring the information backup
  2. An exact copy of a system's information
  3. The ability to get a system u and runnign in the event of a system crash or failure
  4. All of these

Question 434 :
A hard copy of a document is

  1. stored in the hard disk
  2. stored on a floppy disk
  3. stored on a CD
  4. printed on the printer
   printed on the printer

Question 435 :
Which of the following is/are characteristics of Computer?

  1. Diligence
  2. Versatility
  3. Reliability
  4. All of the Above
   All of the Above

Question 436 :
A menu contains a list of

  1. commands
  2. data
  3. objects
  4. reports
   commands

Question 437 :
Which of the following Command is required to reboot the computer

  1. CTRL+ALT+DEL
  2. CTRL+ALT+TAB
  3. CTRL+ALT+SHIFT
  4. CTRL+SHIFT+DEL
   CTRL+ALT+DEL

Question 438 :
________ of cells involves creating a single cell by combining tow or more selected cells.

  1. Formatting
  2. Merging
  3. Embedding
  4. Splitting
   Merging

Question 439 :
Zipping a file means

  1. Encrypting the message
  2. Compressing the message
  3. Transfer the message
  4. All of the above
   Compressing the message

Question 440 :
A file extension is separated from the main file name with a(n) ________, but no spaces.

  1. question mark
  2. exclamation mark
  3. underscore
  4. period
   period

Question 441 :
Firewalls are used to protect against

  1. unauthorised attacks
  2. virus attacks
  3. data driven attacks
  4. fire attacks
   unauthorised attacks

Question 442 :
SMTP, FTP and DNS are applications of the _________ layer.

  1. Data link
  2. Network
  3. Transport
  4. Application
   Application

Question 443 :
What is the correct association between a hardware component and a computer function?

  1. Monitor > input
  2. Mouse > input
  3. CPU > storage
  4. Hard disk > processing
   Mouse > input

Question 444 :
Computers use the ___________ number system to store data and perform calculations.

  1. decimal
  2. hexadecimal
  3. octal
  4. binary
   binary

Question 445 :
A _______ terminal can display images as well as text

  1. text
  2. dumb
  3. graphical
  4. None of the Above
   graphical

Question 446 :
__________ controls the way in which the computer system functions and provides a means by which users can interact with the computer.

  1. The platform
  2. The operating system
  3. Application software
  4. The motherboard
   The operating system

Question 61 :
A _______contains buttons and menus that provide quick access to commonly used commands.

  1. toolbar
  2. status bar
  3. window
  4. find
   toolbar

Question 62 :
What is meant by a dedicated computer ?

  1. Which is used by one person only
  2. Which is assigned one and only one task
  3. Which uses one kind of software
  4. Which is meant for application software
   Which is assigned one and only one task

Question 63 :
A ________ is a special program that processes statements written in a particular programming language and turns theminto machine language

  1. Compiler
  2. Device Driver
  3. Assembler
  4. None of the above
   Compiler

Question 76 :

Question 77 :
Which of the following storage media provides sequential access only?

  2. Magnetic disk

Question 78 :
One advantage of dial-up-internet access is ___

  1. It utilises broadband technology
  2. It utilizes existing telephone service
  3. It uses a router for security
  4. Modem speeds are very fast
   It utilizes existing telephone service

Question 79 :
The display size of a monitor is measured _________

  1. diagonally.
  2. horizontally.
  3. vertically.
  4. None of the Above

Question 80 :
To move to the bottom of a document, press

  1. Insert key

Question 81 :
An assembly language is a division of

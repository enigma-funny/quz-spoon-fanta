Question 391 :
A byte is a collection of _______

  1. 4 bits
  2. 12 bits
  3. 6 bits
  4. 8 bits
   8 bits

Question 392 :
_________ is a generic term for organized collection of computer data and instructions

  1. firmware
  2. Software
  3. hardware
  4. None of the above
   Software

A _______ contains specific rules and words that express the logical steps of an algorithm?

  1. syntax
  2. programming structure
  3. programming language
  4. logic chart
   syntax

Question 394 :
A menu contains a list of

  1. commands
  2. data
  3. objects
  4. reports

Each model of a computer has a unique

  1. Assembly of a computer
  2. Machine language
  3. High level language
  4. All of the above

Question 396 :
The primary output device for computers is a

  1. Video monitor
  2. Printer
  3. Keyboard

Question 397 :
Which of the following is the largest unit of storage?

  1. GB
  2. KB
  3. MB
  4. TB

The part of the CPU which coordinates the activities of all other components of the computer is called

  1. motherboard
  2. coordination board
  3. Arithmetic Logic Unit
  4. control unit

Question 400 :
Technique of using disk space to make programs believe that the system contains more Random Access Memory(RAM) than is actually available is called ______

  1. Random Access Memory
  2. Primary Memory
  3. Secondary Memory
  4. Virtual Memory
   Virtual Memory

Question 401 :
A storage device used to compensate for the differences in rates of flow of data form one device to another is

  1. buffer
  2. channel
  3. call
  4. chip

Question 402 :
What is the name of exel files ?

Question 403 :
Which type of memory is closely related to processor?

  1. Main Memory
  2. SecondaryMemory
  3. Disk Memory
  4. Tape Memory

Question 404 :
Which of the following memories has the shortest access time?

  1. Cache memory
  2. Magnetic Bubble Memory
  3. Magnetic Core Memory
  4. None of the above

In laser printers, printing is achieved by deflecting laser beamon to __________ surface of a drum

Application software

  2. is used to control the operating System

What menu is selected to cut, copy and paste?

  1. File

Question 408 :
Virtual memory is

  1. Memory on the hard disk that the CPU uses as an extended RAM
  2. RAM
  3. Only necessary if you do not have any RAM in your computer
  4. A back up device for floppy disks
   RAM

Question 409 :
LCD stands for _________

  1. Light Crystal Display
  2. Low Crystal Display
  3. Less Crystal Display
  4. Liquid Crystal Display

Question 410 :
Saving is the process of

  1. compying a document from memory to a storage medium
  2. making changes to a document's existing content
  3. changing the appearance, or overall look, of a document
  4. developing a document by entering text using a keyboard

Question 411 :
Website is a collection of :

  1. HTML documents
  2. Graphic files
  3. Audio and Video files
  4. All of the above
   All of the above

Question 412 :
What is the function of Drop Cap ?

  1. It does not allow capital letters to be used in the documents
  2. It makes first letter of each word capital letter in the documents
  3. It lets you begin a paragraph witha large dropped initial capital letter
  4. In automatically starts all paragraphs and sentences with capital letters

Multiprogramming systems

  1. are easier to develop than single programming systems
  2. execute each job faster

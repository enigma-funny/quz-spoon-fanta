Question 361 :
The box that contains the central electronic components of the computer is the

  1. motherboard
  2. system unit
  3. peripheral
  4. input device
   system unit

Question 362 :
Which of the following refers to the fastest, biggest and most expensive computers?

  1. Personal Computers
  2. Supercomputers
  3. Laptops
  4. Notebooks
   Supercomputers

Question 363 :
A _____ shares hardware, software and data among authorised users.

  1. network
  2. protocol
  3. hyperlink
  4. transmitter
   network

Question 364 :
The register which holds the address of the location to or fromwhich data are to be transferred is known as_______

  1. Instruction Register
  2. Control register
  3. Memory Address Register
  4. None of the Above
   Memory Address Register

Question 365 :
_________ are specially designed computer chips that reside inside other devices, such as your car or your electronic thermostat.

  1. Servers
  2. Embedded computers
  3. Robotic computers
  4. Mainframes
   Embedded computers

Question 366 :
It is easy to change the name of file using _____________ process.

  1. transforming
  2. christening
  3. renaming
  4. retagging
   renaming

Question 367 :
The work done by a computer operator is displayed in which part of computer?

  1. CPU
  2. VDU
  3. ALU
  4. None of the Above
   VDU

Question 368 :
________ are dintinct items that don't have much meaning to you in a given context.

  1. Fields
  2. Data
  3. Queries
  4. Properties
   Data

Question 369 :
A walky?talky set is an example of ________ system.

  1. simplex
  2. half duplex
  3. full deplex
  4. double duplex
   half duplex

Question 370 :
Bridge operates in which of the following OSI layers?

  1. Physical
  2. Network
  3. Data link
  4. Both 1) and 3)
   Both 1) and 3)

Question 371 :
The background of any Word document ......

  1. Is always white colour
  2. Is the colour you presect under the Options menu
  3. Is always the same for the entire document
  4. Can have any colour you choose
   Can have any colour you choose

Question 372 :
A storage device or medium where the access time is dependent upon the location of the data is called ________

  1. Parallel access
  2. Serial access
  3. Both (A) and (B)
  4. None of the above
   Serial access

Question 373 :
The Internet is a system of

  1. software bundles
  2. Web pages
  3. Website
  4. Interconnected networks
   Interconnected networks

Question 374 :
Which groupings do you work with when formatting text in Word?

  1. Tables, paragraphs and indexes
  2. Paragraphs, indexes and sections
  3. Characters, sections and paragraphs
  4. Indexes, characters and tables
   Tables, paragraphs and indexes

Question 375 :
______ controls the way in which the computer system functions and provides a means by which users can interact with the computer.

  1. The platform
  2. Application software
  3. Operating system
  4. The motherboard
   Operating system

Question 376 :
_________ printers print the characters without using ribbon and it can print a complete page at a time

  1. Impact
  2. Non Impact
  3. Both (A) and (B)
  4. None of the Above
   Non Impact

Question 377 :
A processor that collects the transmissions from several communications media and sends them over a single line that operates at a higher capacity is called

  1. multiplexor
  2. bridge
  3. Hub
  4. router
   multiplexor

Question 378 :
A device that connects to a network without the use of cables is said to be

  1. distributed
  2. free
  3. centralised
  4. wireless
   wireless

Question 379 :
The ability to find an individual item in a file immediately _____ is used.

  1. File allocation table
  2. Directory
  3. Sequential access
  4. Direct access
   File allocation table

Question 380 :
Compatibility in regard to computers refers to

  1. The software doing the right job
  2. It being versatile enough to handle the job
  3. the software being able to run on the computer
  4. software running with the previously installed software
   software running with the previously installed software

Question 381 :
Which of the following is the newest version of Excel among the following?

  1. Excel 2000
  2. Excel 2002
  3. Excel ME
  4. Excel XP
   Excel XP

Question 382 :
A collection of interrelated files in a computer is a ________.

  1. management information system
  2. spreadsheet
  3. database
  4. text file
   database

Question 383 :
Which of the following is an example of optical disk?

  1. Magnetic disk
  2. Memory disk
  3. Digital Versatile Disk
  4. None of the Above
   Digital Versatile Disk

Question 384 :
Which of the following is/are an example(s) of System Software?

  1. Device Drivers
  2. Language translators
  3. System Utilities
  4. All of the above
   All of the above

Question 385 :
To select an entire row in the Excelsheet, the shortcut key that is used is

  1. Ctrl+Space
  2. Ctrl+Arrow
  3. Shift+Arrow
  4. Shift+Space
   Shift+Space

Question 386 :
Modern computers represent characters and numbers internally using one of the following number systems:

  1. Penta
  2. Octal
  3. Hexa
  4. Binary
   Binary

Question 387 :
The string of eight 0s and 1s is called a

  1. megabyte
  2. kilobyte
  3. byte
  4. gigabyte
   byte

Question 388 :
The blinking symbol on the computer screen is called

  1. mouse
  2. hand
  3. cursor

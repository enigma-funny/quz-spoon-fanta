Question 1291 :
The method of accessing the I/O devices by repeatedly checking the status flags is_________

  1. Memory-mapped I/O
  2. Program-controlled I/O
  3. I/O mapped
  4. None of the Above
   Program-controlled I/O

Question 1292 :
Technique to implement virtual memory where memory is divided into units of fixed size memory is _________

  1. Paging
  2. De-fragments
  3. Segmentation
  4. None of the above
   Paging

Question 1293 :
The ______ may also be called the screen or monitor

  1. Scanner
  2. Display
  3. Hard Disk
  4. None of the Above
   Display

Question 1294 :
Information travels between components on the mother board through

  1. Flash memory
  2. CMOS

Question 1295 :

Question 1296 :
A Proxy server is used for which of the following?

  1. To provide security against unauthorized users
  2. To process client requests for web pages
  3. To provide TCP/IP

Question 1308 :
If a user needs information instantly available to the CPU, it should be stored

  3. in the CPU

Question 1309 :
What is backup?

  1. Adding more components to your network
  2. Protecting data by copying it from the original source to a different destination
  3. Filtering old data from the new data
  4. Accessing data on tape

Question 1310 :

Question 1312 :
Which grouping do you work with when formatting text in word?

Which register keeps tracks of the instructions in the programstored in memory?

Question 691 :
_________ is a device which recognises physical or behavioural traits of the individual

  1. Smart Card Reader
  2. Optical Character Reader(OCR)
  3. Optical Mark Reader(OCR)
  4. Biometric Sensor
   Biometric Sensor

Question 692 :
The transfer of data from a CPU to peripheral devices of computer is achieved through ________

  1. Modem
  2. Interface
  3. Buffer
  4. I/O Ports
   I/O Ports

Question 693 :
Machine code is a

  2. software language
  3. low?level language
  4. middle?level language

Question 694 :
Microcomputer hardware consists of three basic categories of physical equipment:

  1. keyboard, monitor, hard drive
  2. system unit, input/output, memory
  3. system unit, input/output, secondary storage
  4. system unit, primary storage, secondary storage

Question 695 :
Joystick allows movements in _____ directions

  1. Up and Down
  2. Left an Right
  3. Both (A) and (B)
  4. None of the Above
   Both (A) and (B)

Question 696 :
Computers send and receive data in the form of ________ signals.

  1. analog
  2. digital
  3. modulated
  4. demodulated

Question 697 :
The rate at which scanning is repeated in a CRT is called ________

  1. Resolution
  2. Refresh rate
  3. Bandwidth
  4. None of the Above

Question 698 :
The most common input devices are the ________ and ___________.

  2. scanner, board
  3. digital camera, speakers

The portion of the CPU that coordinates the activities of all the other computer components is the

  2. coordination board

Question 702 :
The technique that extends storage capacities of main memory beyond the actual size of the main memory is called

Question 704 :

  3. register mode.

Question 709 :

Question 710 :
To make a notebook act as a desktop model, the notebook can be connected to a ______ which is connected to a monitor and other devices?

Question 711 :
The two basic types of record access methods are _________ and _________.

  1. sequential, random
  2. direct, immediate
  3. sequential, indexed
  4. on?line, realtime

Which groupings do you work with when formatting text in Word?

Question 301 :
A ________ is approximately a million bytes.

  1. gigabyte
  2. kilobyte
  3. megabyte
  4. Terabyte
   megabyte

Question 302 :
In Excel, _________contains one or more worksheets.

  1. Template
  2. Workbook
  3. Active cell
  4. Label
   Workbook

Question 303 :
An example of peripheral equipment is _______

  1. Printer
  2. CPU
  3. Spread Sheet
  4. None of the Above
   Printer

Question 304 :
A ___________ is used to create a digital representation of a printed document or photograph

  1. Video Digitizer
  2. Scanner
  3. Monitor
  4. None of the Above
   Scanner

Question 305 :
Which is an interactive device that facilitates touch sensation and fine-motion control in Robotics and Virtual reality?

  1. Light Pen
  2. Joystick
  3. Data Glove
  4. None of the Above
   Data Glove

Question 306 :
The software tools that enable a user to interact with a computer for specific purposes are known as .......

  1. Hardware
  2. Networked Software
  3. Shareware
  4. Applications
   Applications

Question 307 :
.............. proessing is used when a large mail-order company accumulates orders and processes them together in one large set

  1. Batch
  2. Online
  3. Real-time
  4. Group
   Batch

Question 308 :
SMPT, FTP and DNS are applications of the ________ layer.

  1. Data link
  2. Network
  3. Transport
  4. Application
   Application

Question 309 :
.......... this is the act of copying or downloading a program from a network and making multiple copies of it.

  1. Network piracy
  2. Plagiarism
  3. Software piracy
  4. Site-license piracy
   Software piracy

Question 310 :
The term 'user interface' refers to ............

  1. What the user sees on the screen and how they can interact with it
  2. How the operating system responds to user commands
  3. the means by which the user interacts with the peripheral devices on the computer
  4. the monitor that is available for the computer
   What the user sees on the screen and how they can interact with it

Question 311 :
Which of the following refers to the memory in your computer?

  1. RAM
  2. DSL
  3. USB
  4. LAN
   RAM

Question 312 :
_________ is the transfer of computer data from a temporary storage area to the computer?s permanent memory

  1. Flash
  2. Virtual
  3. Buffer Flush
  4. None of the Above
   Buffer Flush

Question 313 :
The ability of an OS to run more than one application at a time is called

  1. multitasking
  2. object?oriented programming
  3. multi?user computing
  4. time?sharing
   multitasking

Question 314 :
Salami attack is a type of attack that operates on

  1. confidential data
  2. company data
  3. employee data
  4. financial data
   financial data

Question 315 :
Impact printer(s) is/are _______

  1. Dot Matrix printer
  2. Line printer
  3. Daisy Wheel printer
  4. All of the Above
   All of the Above

Question 316 :
The internet is ..........

  1. a large network of networkds
  2. an internal communication system for a business
  3. a communications system for the Indian government
  4. All of these
   a large network of networkds

Question 317 :
Which of the following peripheral devices displays information to a user?

  1. Monitor
  2. Keyboard
  3. Secondary storage device
  4. Scanner
   Monitor

Question 318 :
______ is RAM that combines the fast read and write access of Dynamic RAM

  1. VRAM
  2. WRAM
  3. MRAM
  4. FRAM
   FRAM

Question 319 :
What are .bas, .doc and .htm examples of?

  1. Databases
  2. Extensions
  3. Domains
  4. Protocols
   Extensions

Question 320 :
This component is required to process data into information and consists of integrated circuits.

  1. Hard disk
  2. RAM
  3. CPU
  4. ROM
   CPU

Question 321 :
Personal computers are connected to form

  1. Super computer
  2. Mainframe computers
  3. Enterprise
  4. Network
   Network

Question 322 :
A(n) ________ is the text you want to be printed at the bottom of the pages.

  1. header
  2. endnote
  3. footnote
  4. footer
   footer

Question 323 :
_________ is a group of commands for a CPU in machine language

  1. Information Set
  2. Instruction Set
  3. Buffer
  4. None of the Above
   Instruction Set

Question 324 :
What is the keyboard shortcut for creating chart from the selected cell range?

  1. F2
  2. F4
  3. F8
  4. F11
   F11

Question 325 :
The bar at the top of a window that bears the name of the window is known as?

  1. control panel
  2. task bar
  3. menu bar
  4. title bar
   title bar

Question 326 :
A _________ helps in converting programming languages to machine language

  1. Operating system
  2. System Utilities
  3. Language translator
  4. Application software
   Language translator

Question 327 :
Data going into the computer is called?

  1. Output
  2. Algorithm
  3. Input
  4. Calculations
   Input

Question 328 :
The primary purpose of software is to turn data into?

  1. Web sites
  2. information
  3. programs
  4. objects
   information

Question 329 :
What is output ?

  1. What the processor takes from the user
  2. What the user gives to the processor
  3. What the processor gets from the user
  4. What the processor gives to the user

Question 330 :
A ______ is a design tool that graphically shows the logic in a solution algorithm.

  2. hierarchy chart

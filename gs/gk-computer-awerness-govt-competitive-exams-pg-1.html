Question 1 :
Which part of the computer helps in storing information?

  1. Disk drive
  2. Keyboard
  3. Monitor
  4. Printer
   Disk drive

Question 2 :
ADSL data communications technology uses which of the following for faster data transmission ?

  1. Voiceband modem
  2. Wireless Modem
  3. Copper telephone Lines
  4. Sockets
   Copper telephone Lines

Question 3 :
This type of hardware consists of devices that translate data into a form the computer can process

  1. application
  2. input
  3. system
  4. All of these
   input

Question 4 :
Instructions and data can be stored in the memory of Computer for automatically directing the flow of operations

  1. Objective Programming
  2. Stored program
  3. Both (A) and (B)
  4. None of the Above
   Stored program

Question 5 :
What is the shortcut key to Undo the last action in a document?

  1. Ctrl + X
  2. Ctrl + Y
  3. Ctrl + Z
  4. Ctrl + U
   Ctrl + Z

Question 6 :
How many different documents can you have open at any one time?

  1. Not more than three
  2. Only one
  3. As many as your computer memory will hold
  4. No more than your task bar can display
   As many as your computer memory will hold

Question 7 :
What is the shortcut key for 'Find and Replace' dialogue box?

  1. Ctrl+A
  2. Ctrl+R
  3. Ctrl+H
  4. Ctrl+M
   Ctrl+H

Question 8 :
A ________ is a set of rules.

  1. resource locator
  2. domain
  3. hypertext
  4. protocol
   protocol

Question 9 :
Which of the following is an example of connectivity?

  1. Internet
  2. Floppy disk
  3. paper disk
  4. TV disk
   Internet

Question 10 :
____________ is a collection of one or more files that correct flaws in the performance, reliability or security of a specific software product

  1. Software Update
  2. Software Piracy
  3. Software patch
  4. None of the above
   Software patch

Question 11 :
Which is the most popular medium for direct access to secondary storage of a computer?

  1. Magnetic tape
  2. ROM
  3. RAM
  4. Magnetic disk
   Magnetic disk

Question 12 :
The computer code for interchange of information between terminals is

  1. BCDIC
  2. BCD
  3. ASCII
  4. None of the above
   ASCII

Question 13 :
All of the following storage media have Read and Write capabilities except

  1. flash memory cards
  2. CD?ROMs
  3. hard disk drives
  4. floppy disks
   CD?ROMs

Question 14 :
You must install a(n) ________ on a network if you want to share a broadband Internet connection.

  1. router
  2. modem
  3. node
  4. cable
   router

Question 15 :
Which key is used in combination with another key to perform a specific task?

  1. Function
  2. Arrow
  3. Space bar
  4. Control
   Control

Question 16 :
Trackball is an example of a/an _____

  1. Output device
  2. Printing device
  3. Pointing device
  4. None of the Above
   Pointing device

Question 17 :
........... is the science that attempts to produce machines that display the same type of intelligence that humans do

  1. nanoscience
  2. Nanotechnology
  3. Simulation
  4. Artificial intelligence
   Artificial intelligence

Question 18 :
For seeing the output, which of the following is used?

  1. Scanner
  2. Keyboard
  3. Mouse
  4. Monitor
   Monitor

Question 19 :
Ethernet uses _________ topology.

  1. bus
  2. ring
  3. star
  4. tree
   bus

Question 20 :
It is the set of programs that enables your computers hardware device and application software to work together

  1. Operating system
  2. Helper software
  3. System software
  4. Application software
   System software

Question 21 :
Main memory works in conjunction with

  1. special function cards
  2. RAM
  3. CPU
  4. Intel
   CPU

Question 22 :
Softloading is also called __________

  1. End User Piracy
  2. Softlifting
  3. Both (A) and (B)
  4. None of the above.
   Both (A) and (B)

Question 23 :
Which of the following memories uses a MOS capacitor as its memory cell?

  1. SRAM
  2. DRAM
  3. ROM
  4. FIFO
   DRAM

Question 24 :
What menu is selected to cut, copy and paste?

  1. File
  2. Edit
  3. Tools
  4. Table
   File

Question 25 :
Computers connected to a LAN can

  1. run faster
  2. go online
  3. share information and/or share peripheral equipment
  4. e?mail
   share information and/or share peripheral equipment

Question 26 :
The errors that can be pointed out by the compiler are?

  1. Syntax errors
  2. Semantic errors
  3. Logic errors
  4. System errors
   Syntax errors

Question 27 :
Which is a hardware component or system of components that allows a human being to interact with a computer?

  1. Interface device (IDF)
  2. Graphical User Interface (GUI)
  3. Musical Digital Instrument Interface (MIDI)
  4. None of the Above
   Interface device (IDF)

Question 28 :
The move to the bottom of a document, press ........

  1. Auto summarize
  2. Home key
  3. Ctrl + Home key
  4. Ctrl + End key
   Ctrl + End key

Question 29 :
Array is ............

  1. linear data structure
  2. non-linear data structure
  3. complex data structure
  4. simple data structure
   linear data structure

Question 30 :
A collection of unprocessed items is

  1. information
  2. data
  3. memory
  4. reports
   data

Question 871 :
Given here are four places situated on the banks of the Narmada :1. Jabalpur 2. Mandla 3. Amarkantak 4. Bharuch - In which order do these places occur from the source of the river to its mouth?

  1. 3, 2, 1,4
  2. 2, 1, 4, 3
  3. 1, 4, 3, 2
   3, 2, 1,4

Question 872 :
Which, among the following countries, occupies the largest chunk of the Sunderban's Delta?

  1. Bangladesh
  2. Myanmar
  3. India
  4. Thailand
   Bangladesh

Question 873 :
The ______ route is the busiest ocean route of the world.

  1. Suez
  2. Panama
  3. North Atlantic
  4. South Atlantic
   North Atlantic

Question 874 :
For the formation of rainfall which of the following is not essential?

  1. dust particles in the atmosphere
  2. evaporation
  3. condensation
  4. wind
   wind

Question 875 :
River Godavari flows through: [NDA 1994]

  1. Maharashtra and Andhra Pradesh
  2. Maharashtra, Orissa and Andhra Pradesh
  3. Maharashtra, Karnataka and Andhra Pradesh
  4. Maharashtra, Karnataka, Orissa and Andhra Pradesh
   Maharashtra and Andhra Pradesh

Question 876 :
The Sivasamudram Falls is on : [Teachers' Exam 1994]

  1. Cauvery
  2. Subernarekha
  3. Indravati
  4. None of these
   Cauvery

Question 877 :
Narmada river originates from Amarkantak in:

  1. Gujarat
  2. Maharashtra
  3. Madhya Pradesh
  4. Chhattisgarh
   Madhya Pradesh

Question 878 :
Which of the following rivers joins Ganga at Allahabad ? [RRB 1991]

  1. Gandak
  2. Ghaghra
  3. Kosi
  4. Yamuna
   Yamuna

Question 879 :
Canal irrigation is most important in the Northern Plains of India because:

  1. the soil is porous
  2. the level of underground water is high
  3. the sources of canals are perennial rivers
  4. the region is densely populated
   the sources of canals are perennial rivers

Question 880 :
Which are the west-flowing rivers of southern Indian which flow into Arabian Sea? [Teachers' Exam 1991]

  1. Chambal and Betwa
  2. Damodar and Mahanadi
  3. Narmada and Tapti
  4. Krishna and Cauveri
   Narmada and Tapti

Question 881 :
Which of the following rivers meet Ganga in Bihar ? 1. Ghaghra 2. Gandak 3. Son 4. Punpun [PCS 1995]

  1. 1, 2 and 4
  2. 1, 3 and 4
  3. 2 and 4
  4. 1, 2, 3 and 4
   1, 2, 3 and 4

Question 882 :
Shwedagon pagoda is located in

  1. Mahabalipuram
  2. Rangoon
  3. Mandalay
  4. Jakarta
   Rangoon

Question 883 :
Which of the following is taken into consideration while deciding a shipping route?

  1. Depth of the sea
  2. Availability of ports and harbours
  3. Climate of the region
  4. All the above
   All the above

Question 884 :
Haneda is the name of the international airport in

  1. Hong Kong
  2. Dacca
  3. Tokyo
  4. Singapore
   Tokyo

Question 885 :
Arrange the following rivers in descending order of their length: 1. Ravi 2. Cauvery 3. Sutlej 4. Yamuna

  1. 1, 2, 4, 3
  2. 3, 4, 1, 2
  3. 4, 3, 2, 1
  4. 1, 2, 3, 4
   4, 3, 2, 1

Question 886 :
The port especially developed for exporting iron ore to Japan is: [CDS 2002]

  1. Haldia
  2. New Tuticorin
  3. Paradeep
  4. Vishakhapatnam
   Paradeep

Question 887 :
Which one of the following pairs of sea ports and states is not correctly matched?

  1. Kandla : Gujarat
  2. Paradip : Orissa
  3. Quilon : Kerala
  4. Kakinada : Maharashtra
   Kakinada : Maharashtra

Question 888 :
Which of the following lakes were formed by volcanic activities? [IAS 2003] (A) Lonar lake (B) Bhim Tal of Kumaun (C) Toba lake (D) Keloed lake

  1. A and C
  2. A and D
  3. A, C and D
  4. A, B, C and D
   A, C and D

Question 889 :
The Indian riven of the Indus Basin are

  1. Sutlej, Beas and Ravi
  2. Ganga, Beas and Sutlej
  3. Betwa, Sone and Kosi
  4. Gomati, Kosi, Gandak
   Sutlej, Beas and Ravi

Question 890 :
Which of the following rivers flows between the Vindhya and Satpura ranges?

  1. Narmada
  2. Tapti
  3. Gandak
  4. Godavari
   Narmada

Question 891 :
Ankaleshwar in India is known for the production of: [NDA 2002]

  1. bauxite
  2. coal
  3. iron ore
  4. petroleum
   petroleum

Question 892 :
The main river flowing in the state of Jammu and Kashmir is :

  1. Jhelum
  2. Indus
  3. Chenab
  4. Nubra
   Jhelum

Question 893 :
Which one of the following is a north flowing river?

  1. Cauvery
  2. Narmada
  3. Chambal
  4. Brahmaputra
   Narmada

Question 894 :
Which of the following river is known as India's River of Sorrow? [UDC 1993]

  1. Hooghly
  2. Damodar
  3. Ghaghara
  4. Kosi
   Kosi

Question 895 :
Which one of the following gives the correct sequence of ports down the western coast of India?

  1. Kandla, Mumbai, Kochi, Mangalore
  2. Porbandar, Marmagao, Mangalore, Kochi
  3. Veraval, Okha, Marmagao, Mangalore
  4. Mangalore, Marmagao, Kochi, Negercoil
   Porbandar, Marmagao, Mangalore, Kochi

Question 896 :
The highest waterfall in India is : [PCS 1994]

  1. Sivasundaram
  2. Jog
  3. Courtallam
  4. Hogenakal
   Jog

Question 897 :
Horizontal transfer of heat is known as

  1. advection
  2. convection
  3. conduction
  4. elliptical line
   advection

Question 898 :
The 'land of eternal spring' are certain parts of

  1. Tundra region
  2. Highland region
  3. Mediterranean region
  4. Hot Wet Equatorial region
   Hot Wet Equatorial region

Question 899 :
Which one of the following lakes in India has the highest Water salinity? [NDA 1994]

  1. Dal
  2. Chilka
  3. Wular
  4. Sambhar
   Sambhar

Question 900 :
Consider the following rivers: 1. Betwa 2. Kosi 3. Gandak. - Which of the above joint(s) Yamuna river? [NDA 2008]

  1. 1 only
  2. 1 and 2
  3. 1 and 3
  4. 2 and 3
   2 and 3

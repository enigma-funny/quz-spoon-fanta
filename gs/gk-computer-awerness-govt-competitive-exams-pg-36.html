Question 1051 :
Which of the following are all considered advantages of e?mail?

  1. Convenience, speed of delivery, generality and reliability
  2. Printable, global and expensive
  3. Global, convenience and Microsoft owned
  4. Slow delivery, reliable, global and inexpensive
   Convenience, speed of delivery, generality and reliability

Question 1052 :
Band, the unit for measuring data transmission speed, is equal to

  1. one bit per second
  2. one byte per second
  3. two bytes per second
  4. two bits per second
   one bit per second

Question 1053 :
To what temporary area can you store text and other data, and later paste them to another location?

  1. The clipboard
  2. ROM
  3. CD-ROM
  4. The hard disk
   The clipboard

Question 1054 :
Test the validity of the statements ?

  1. 1 KB = 1024 BYTES
  2. 1 MB = 2048 BYTES
  3. 1MB = 100 KILOBYTES
  4. 1 KB = 1000 BYTES
   1 KB = 1024 BYTES

Question 1055 :
Which of the following produces the best quality graphics reproduction?

  1. Laser printer
  2. Inkjet printer
  3. Plotter
  4. Dot matrix printer
   Plotter

Question 1056 :
Storage device where time to retrieve stored information is independent of address where it is stored is called _______

  1. Random Access Memory
  2. Secondary Memory
  3. System
  4. None of the above
   Random Access Memory

Question 1057 :
A group of 8 bits is known as

  1. byte
  2. kilobyte
  3. binary digit
  4. megabit
   byte

Question 1058 :

  1. Smart Card Reader
  2. Optical Character Reader(OCR)
  3. Optical Mark Reader(OCR)
  4. Biometric Sensor
   Biometric Sensor

Question 1059 :
Example of non?numeric data is

  4. None of these

Question 1060 :
Computers send and receive data in the form of ________ signals.

  1. analog
  2. digital
  3. modulated
  4. demodulated

Question 1061 :
Language Translators can be divided into three major categories

  1. Compiler, Operating System and Assembler
  2. Compiler, Device Driver and Assembler
  3. Compiler, Interpreter and System Utility
  4. Compiler, Interpreter and Assembler

Question 1062 :
Where are programs and data kept while the processor is using them?

  1. Main memory
  2. Secondary memory
  3. Disk memory
  4. Program memory

Question 1063 :
How are data organized in a spreadsheet?

  1. Lines and spaces
  2. Layers and planes
  3. Height and width

The acronym HTML stands for

  1. High Transfer Machine Language
  2. High Transmission Machine Language
  3. Hyper Text Markup Language

Question 1065 :
Which of the following memory chip is faster?

  1. DRAM

  3. Both (A) and (B)

Question 1067 :
If you do not want to select any option after opening a menu then click menu title again or press key ________ to close the menu.

  1. Shift
  2. Tab
  3. Escape
  4. F1
   Escape

Question 1068 :
Programs such as Internet Explorer that serve as navigable windows into the Web are called

  1. Hypertext
  2. Networks
  3. Internet
  4. Web browsers
   Web browsers

Question 1069 :
Which is an escape code language used to send commands to the printer for printing documents?

  1. Postscript
  2. PCL
  3. Both (A) and (B)
  4. None of the Above
   PCL

Question 1070 :
Select the ODD one

  1. Operating system
  2. Interpreter
  3. Compiler

Question 1071 :
Every component of your computer is either

  1. software or CPU/RAM
  2. input devices or output devices

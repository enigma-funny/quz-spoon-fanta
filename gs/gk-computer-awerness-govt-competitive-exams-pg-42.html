Question 1231 :
Which neural network allows feedback signal?

  1. Feed forward Neural Network
  2. Recurrent Neural Network
  3. Both(A) - (B)
  4. None of the Above
   Recurrent Neural Network

Question 1232 :
Computers use the _____ number system to store data and perform calculations.

  1. Decimal
  2. Hexadecimal
  3. Octal
  4. Binary
   Binary

Question 1233 :
The Internet is an example of __________ network.

  1. star
  2. bus
  3. hybrid
  4. mesh
   mesh

Question 1234 :
When the pointer is positioned on a _____ it is shaped like a hand.

  1. Grammar error
  2. Hyperlink
  3. Screen tip
  4. Spelling error
   Hyperlink

Question 1235 :
The property that, when classes are arranged in a hierarchy, each class assumes the attributes and methods of its ancestores is

  1. Inheritance
  2. generalisation
  3. encapsulation
  4. information hiding
   Inheritance

Question 1236 :
A computer does not suffer from tiredness and lack of concentration

  1. Diligence
  2. Versatility
  3. GIGO
  4. None of the Above

Question 1237 :

Question 1238 :
Which of the following functions is NOT performed by the TCP?

  1. Addressing
  2. Connection establishment
  3. Data transfer
  4. None of these

Question 1239 :
In which addressing mode the operand is given explicitly in the instruction?

  1. Absolute
  2. Immediate
  3. Indirect
  4. Direct
   Immediate

Question 1240 :
The toner or ink in a Laser printer is________

  1. dry
  2. wet
  3. Either (A) or (B)
  4. None of the Above
   dry

Question 1241 :
All the information collected during database development is stored in a

  1. repository
  2. dataware house
  3. RAD

Question 1242 :
When we run a program in HTML coding _____ is used as backend and ____ works as front-end.

  1. Notepad - Internet-Explorer
  2. Notepad - Msn-messenger
  3. MS-Word - Internet Explorer
  4. Both (1) and (2)
   Notepad - Internet-Explorer

Question 1243 :
What is a firmware?

  1. A program started in RAM.
  2. A program started in a file.
  3. A program started in boot sector.
  4. A program started in ROM.

Which shortcut key is used to open a new file in MS Word?

  1. Ctrl+N

Question 1247 :
When necessary, the results are transferred from the CPU to main memory by _______

  1. I/O devices.
  2. CPU.
  3. Shift registers.
  4. None of the Above.

Machine code is a

Question 1249 :
A collection of unprocessed items is

  1. Information

Question 1250 :
Which of the following is Not a characteristic of E-mail ?

  1. Low cost
  2. Record maintenance in database
  3. High speed

Question 1251 :
What is the name of the protocol used to eliminate loops?

Question 1321 :
Which area in an Excel window allows entering values and formulas?

  1. Title bar
  2. Menu bar
  3. Formula bar
  4. Standard Tool Bar
   Formula bar

Question 1322 :
Which of the following is NOT an output device?

  1. Plotter
  2. Printer
  3. Scanner
  4. Monitor
   Scanner

Question 1323 :
Joystick allows movements in _____ directions

  1. Up and Down
  2. Left an Right
  3. Both (A) and (B)
  4. None of the Above
   Both (A) and (B)

Question 1324 :
A saved document is referred to as a

  1. file
  2. word
  3. folder
  4. project
   file

Question 1325 :
Select the odd one out :

  1. Interpreter
  2. Operating system
  3. Compiler
  4. Assembler
   Operating system

Question 1326 :
Portable computer, also known as laptop computer, weighing between 4 and 10 pounds is called

  1. general?purpose application
  2. Internet
  3. scanner
  4. notebook computer
   notebook computer

Question 1327 :
Microsoft Office is an example of a

  1. closed?source software
  2. open?source software
  3. horizontal?market software
  4. vertical?market software
   closed?source software

Question 1328 :
Transformation of input into output is performed by?

  1. Peripherals
  2. Memory
  3. Storage
  4. The CPU
   The CPU

Question 1329 :
After a picture has been taken with a digital camera and processed appropriately, the actual print of the picture is considered a(n)

  1. data
  2. output
  3. input
  4. process
   output

Question 1330 :
Bold, Itali, Regular are known as

  1. font styles
  2. font effects
  3. word art
  4. text effects
   font styles

Question 1331 :
Personal logs or journal entries posted on the Web are known as :

  1. listservs
  2. Webcasts
  3. blogs
  4. subject directories
   blogs

Question 1332 :
Text and graphics that have been cut or copied are stored in an area called the

  1. Pasterboard
  2. Copyboard
  3. Clipboard
  4. Cuttingboard
   Clipboard

Question 1333 :
The interface of data from a CPU to peripheral devices of a computer is achieved through

  1. interface
  2. commands
  3. buffer
  4. parts of computer
   parts of computer

Question 1334 :
What menu is selected to cut, copy and paste?

  1. File
  2. Tools
  3. Special
  4. Edit
   Edit

Question 1335 :
The control unit controls other units by generating control and _______

  1. Command Signals
  2. Timing signals
  3. Transfer signals
  4. None of the Above
   Timing signals

Question 1336 :
The following are all computing devices, except?

  1. notebook computers
  2. cellular telephones
  3. digital scanners
  4. personal digital assistants
   digital scanners

Question 1337 :
How can the user determine what programs are available on a computer?

  1. Checking the hard disk properties
  2. Viewing the installed programs during the booting process
  3. Checking the operating system for a list of installed programs.
  4. Checking the existing files saved on the disk
   Checking the existing files saved on the disk

Question 1338 :
___________ is the process of carrying out commands.

  1. Fetching
  2. Storing
  3. Decoding
  4. Executing
   Executing

Question 1339 :
A computer is a box full of electronic

  1. chips
  2. registers
  3. circuits
  4. switching devices
   switching devices

Question 1340 :
When you are working on a document on PC, where is the document temporarily stored?

  1. RAM
  2. ROM
  3. The CPU
  4. Flash memory
   RAM

Question 1341 :
Which of the following storage media provides sequential access only?

  1. Floppy disk
  2. Magnetic disk
  3. Magnetic tape
  4. Optical disk

Question 641 :
_________is a copy of Basic Input/Output Operating System (BIOS) routines from Read Only Memory (ROM) into a special area of RAM so that they can be accessed more quickly

  1. Dynamic RAM
  2. Shadow RAM
  3. Synchronous Graphics RAM
  4. Video RAM

Question 647 :
The WWW standard allows programs on many different computer platforms to show the information on a server. Such programs are called :

  1. ISP
  2. Web Browsers
  3. Web Servers
  4. Web Modular

Question 648 :
The ________ , also called the Web, contains billions of documents.

  1. Web portal
  2. HTTP
  3. World Wide Web
  4. Domain

Question 649 :
The term ________ refers to data storage systems that make it possible for a computer or electronic device to store and retrieve data

  2. output technology
  3. storage technology
  4. None of the Above

Question 650 :

An integrated circuit is

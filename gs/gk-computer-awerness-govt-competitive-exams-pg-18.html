Question 511 :
The first computers were programmed using

  1. assembly language
  2. machine language
  3. spaghetti code
  4. source code
   machine language

Question 512 :
The first computer language developed was______

  1. COBOL
  2. PASCAL
  3. BASIC
  4. FORTRAN
   FORTRAN

Question 513 :
Impact printers can be divided into ______ types

  1. Four
  2. Six
  3. Three
  4. Two
   Two

Question 514 :
Which of the following bus structure is usually used to connect I/O devices?

  1. Single bus
  2. Multiple bus
  3. Star bus
  4. None of the Above
   Single bus

Question 515 :
Father of modern computer______

  1. Charles Babbage
  2. Alan Turing
  3. Ted Hoff
  4. None of the Above
   Alan Turing

Question 516 :
How many types of data can be entered in a worksheet?

  1. Three
  2. Four
  3. Two
  4. Five
   Three

Question 517 :
Synonym can be reviewed with

  1. Spelling and grammar
  2. Thesaurus
  3. Both
  4. Synonym viewer
   Thesaurus

Question 518 :
Virtual memory is

  1. memory on the hard disk that the CPU uses as extended RAM
  2. in RAM
  3. only necessary if you do not have any RAM in your computer
  4. a backup device for floppy disks
   memory on the hard disk that the CPU uses as extended RAM

Question 519 :
.....is collection of web pages and ......is the very first page that we see on opening of a web-site

  1. Home-page, Web-page
  2. Web-site, Home-page
  3. Web-page, Home-page
  4. Web-page, Web-site
   Web-site, Home-page

Question 520 :
Changing an existing document is called _____ the document.

  1. Creating
  2. Editing
  3. Modifying
  4. Adjusting
   Editing

Question 521 :
Which part of the computer displays the work done?

  1. RAM
  2. Scanner
  3. Monitor
  4. ROM
   Monitor

Question 522 :
A group of bits that tell the computer to perform a specific operation is known as_______

  1. Instruction code
  2. Micro-operation
  3. Accumulator
  4. Register
   Instruction code

Question 523 :
Workgroup means

  1. Computers in network
  2. Individual user
  3. Individual computer
  4. All of he above
   Computers in network

Question 524 :
A(n) ___________ is a small, wirless handheld computer that scans items, tags and pulls up the current price (and any special offer) as you shop.

  1. PSS
  2. PoS
  3. Inventary
  4. Data mining
   PSS

Question 525 :
A disk?s content that is recorded at the time of manufacture and that cannot be changed or erased by the user is ________

  1. Write only
  2. Read Only
  3. Both (A) and (B)
  4. None of the Above
   Read Only

Question 526 :
A(n) ________ is the combination of software and hardware that facilitates the sharing of information between computing devices.

  1. network
  2. peripheral
  3. expansion board
  4. digital device
   network

Question 527 :
Which function calculates the largest value in a set of numbers?

  1. Average
  2. Count
  3. Minimum
  4. Maximum
   Maximum

Question 528 :
Chip is a common nickname for a(n)?

  1. transistor
  2. resistor
  3. integrated circuit
  4. semiconductor
   integrated circuit

Question 529 :
Which type of protocol is used for accessing Internet/web on mobile phones?

  1. PPP
  2. HTTP
  3. FTP
  4. WAP
   WAP

Question 530 :
In a relational database, this is a data structure that organises the information about a single topic into rows and columns.

  1. block
  2. record
  3. tuple
  4. table
   table

Question 531 :
EDODRAM stands for _________

  1. Extended Digital Output Dynamic RAM
  2. Extended Dynamic Output Digital RAM
  3. Extended Data Output Digital RAM
  4. Extended Data Output Dynamic RAM
   Extended Data Output Dynamic RAM

Question 532 :
The parity bit is added for

  1. loading
  2. indexing
  3. updating
  4. error detection
   error detection

Question 533 :
Where are you likely to find an embedded operating system?

  1. On a desktop operating system
  2. on a networked PC
  3. on a network server

Question 535 :
External Bus is also referred as _________

Question 871 :
A CPU contains

  1. a card reader and a printing device
  2. an analytical engine and a control unit
  3. a control unit and an arithmetic logic unit
  4. an arithmetic logic unit and a card reader
   a control unit and an arithmetic logic unit

Question 872 :
Any data or instructions sent to the computer is termed as

  1. storage
  2. output
  3. input
  4. information
   input

Question 873 :
The ________ becomes different shapes depending on the task you are performing.

  1. active tab
  2. insertion point
  3. mouse pointer
  4. ribbon
   mouse pointer

Question 874 :
The operating system called UNIX is typically used for

  1. desktop computers
  2. laptop computers
  3. super computers
  4. All the above
   All the above

Question 875 :
Which of the following is NOT a part of the operating system?

  1. Supervisor
  2. Jon?control program
  3. I/O control program
  4. Performance monitor
   Performance monitor

Question 876 :
Only ________program(s) remain(s) active even though we can open many programs at a time.

  1. one
  2. two
  3. three
  4. four
   one

Question 877 :
The Internet is an example of __________ network.

  1. star
  2. bus
  3. hybrid
  4. mesh
   mesh

Question 878 :
Which of the following is System Software?

  1. Microsoft Word
  2. Microsoft Excel
  3. Adobe Photoshop
  4. Windows 7
   Windows 7

Question 879 :
Which of the following was the first widely used OS with graphical interface?

  1. OS12
  2. MVS
  3. Mac OS
  4. DOS
   Mac OS

Question 880 :
A displaying screen in which text is presented in one colour and background is of any other colour is called ________

  1. monochrome screen
  2. high resolution screen
  3. low resolution screen
  4. medium resolution screen
   monochrome screen

Question 881 :
Software designed for a specific purpose is called

  1. General Program
  2. Specific Program
  3. Application Software
  4. System Software
   Application Software

Question 882 :
Storage that retains its data after the power is turned off is referred to as?

  1. volatile storage
  2. non-volatile storage
  3. sequential storage
  4. direct storage
   non-volatile storage

Question 883 :
Office LANs that are spread geographically apart on a large scale can be connected using a corporate _____

  1. CAN
  2. LAN
  3. DAN
  4. WAN
   WAN

Question 884 :
You can use the tab key to

  1. Move a cursor across the screen
  2. Indent a paragraph
  3. Move the cursor down the screen
  4. Only (1) and (2)
   Indent a paragraph

Question 885 :
Computer programs are written in a high?level programming language? however, the human?readable version of a program is called

  1. cache
  2. instruction set
  3. source code
  4. world size
   source code

Question 886 :
Select the odd man out.

  1. Interpreter
  2. Operating system
  3. Compiler
  4. Assembler
   Operating system

Question 887 :
What menu is selected to cut, copy, and paste?

  1. File
  2. Tools
  3. Special
  4. Edit
   Edit

Question 888 :
Unsolicited commercial email is commonly known as

  1. spam
  2. junk
  3. hoaxes
  4. hypertext
   spam

Question 889 :
Doing research by using an on-line database instead of a printed set of similar information offers which of the following advantage

  1. More current information
  2. Easier to understand
  3. Better charts
  4. None of the above
   More current information

Question 890 :
Which of the following is not an output device?

  1. Plotter
  2. Printer
  3. Monitor
  4. Joystick
   Joystick

Question 891 :
The blinking point which shows your position in the text is called

  1. blinker
  2. cursor
  3. causer
  4. pointer
   cursor

Question 892 :
A _____ contains specific rules and words that express the logical steps of an algorithm.

  1. Syntax
  2. Programming structure
  3. Programming language
  4. Logic chart
   Programming language

Question 893 :
__________ is an interface for connecting eight or more data wires

  1. Serial Port
  2. Fire wire
  3. Parallel Port
  4. None of the Above
   Parallel Port

Question 894 :
Network components are connected to the same cable in the .......... topology

  1. star
  2. ring
  3. bus
  4. mesh
   star

Question 895 :
We access the World Wide Web using :

  1. Browsers
  2. Instant messaging applications
  3. High bandwidth
  4. Search engine
   Browsers

Question 896 :
To access properties of an object, the mouse technique to use is __________.

  1. dragging
  2. dropping
  3. right?clicking
  4. shift?clicking
   right?clicking

Question 897 :
A computer is a box full of electronic

  1. chips
  2. registers
  3. circuits
  4. switching devices
   switching devices

Question 898 :
What term applies to a collection of related records in a database?

  1. Field
  2. File
  3. Both (a) and (b)
  4. None of the above
   File

Question 899 :
The sharing of a medium and its path by 2 or more devices is called ..........

  1. Modulation
  2. Encoding
  3. Line discipline
  4. Multiplexing
   Multiplexing

Question 900 :
What is output?

  1. What the processor takes from the user.
  2. What the user gives to the processor.
  3. What the processor gets from the user.
  4. What the processor gives to the user.
   What the processor gives to the user.

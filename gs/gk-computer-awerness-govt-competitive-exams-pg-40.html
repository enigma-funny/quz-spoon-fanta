VDU stands for________

  1. Video Display Unit
  2. Visual Display Unit
  3. Video Divide Unit
  4. None of the Above

Question 1182 :
Every computer connected to an intranet or extranet must have a distinct .......

  1. firewall
  2. proxy server
  3. IP address
  4. domain name
   proxy server

Question 1183 :
_________is computer storage that only maintains its data while the device is powered

  1. Volatile memory
  2. Non Volatile memory
  3. Both (A) and (B)
  4. None of the Above
   Volatile memory

The part of CPU which coordinated the activities of all other components of the computer is

  1. motherboard
  2. coordination board
  3. Arithmetic Logic Unit

The portion of the CPU that coordinates the activities of all the other computer components is the

  1. motherboard
